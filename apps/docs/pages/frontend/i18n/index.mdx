## What is i18n?

Internationalization (i18n) is the process of designing and developing your application to support multiple languages and regions. Our frontend uses **next-intl** to provide a seamless multi-language experience.

## Current Support

- **English (en)** - Default language
- **Spanish (es)** - Fully translated

## Key Features

✅ **Automatic locale detection** from browser settings  
✅ **Server-side rendering** with localized content  
✅ **Type-safe** translation keys  
✅ **SEO-friendly** URLs with locale prefixes  
✅ **Easy language switching** with built-in component

## Quick Example

```tsx
import { useTranslations } from "next-intl";

export default function MyComponent() {
  const t = useTranslations();

  return (
    <div>
      <h1>{t("common.welcome")}</h1>
      <p>{t("auth.loginTitle")}</p>
    </div>
  );
}
```
