# Getting Started

## Installation

The toast system requires **Jotai** to be installed in the frontend app. **Sonner** is already included in the UI package dependencies.

### Step 1: Install Dependencies

```bash
cd apps/frontend
pnpm install
```

This will install Jotai (already added to `package.json`).

### Step 2: Rebuild UI Package

```bash
cd ../../packages/ui
pnpm run build
```

### Step 3: Restart Development Server

```bash
cd ../../apps/frontend
pnpm run dev
```

## Verification

The toast system is already integrated in the app through the providers. Verify it's working by navigating to `/toast-test` in your browser and clicking the test buttons.

## Quick Start

Once installed, you can use the toast system in any component:

```tsx
import { useGlobalToast } from "@/hooks";

function MyComponent() {
  const toast = useGlobalToast();

  const handleClick = () => {
    toast.success("It works!");
  };

  return <button onClick={handleClick}>Show Toast</button>;
}
```

## Integration

The toast system is already integrated in your application through `apps/frontend/app/providers.tsx`:

```tsx
import { Provider as JotaiProvider } from "jotai";
import { ToastProvider } from "@/components/ToastProvider";

export function Providers({ children }) {
  return (
    <JotaiProvider>
      <QueryClientProvider client={queryClient}>
        <AuthProvider>
          <ToastProvider />
          {children}
        </AuthProvider>
      </QueryClientProvider>
    </JotaiProvider>
  );
}
```

No additional setup is required!

## Testing Your Installation

### Option 1: Use the Demo Page

Navigate to `/toast-test` in your browser to see an interactive demo with all toast types and features.

### Option 2: Add to an Existing Page

```tsx
"use client";

import { useGlobalToast } from "@/hooks";
import { Button } from "@project-template/ui";

export default function TestPage() {
  const toast = useGlobalToast();

  return (
    <div className="p-8 space-y-4">
      <Button onClick={() => toast.success("Success!")}>Success Toast</Button>
      <Button onClick={() => toast.error("Error!")}>Error Toast</Button>
      <Button onClick={() => toast.info("Info!")}>Info Toast</Button>
    </div>
  );
}
```
